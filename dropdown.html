Usage

Dropdown is created by a div element that has the dropdown class and a menu created with a div that has the ui drop menu classes.

To link the dropdown with the menu, The menu should be created inside the dropdown div created earlier or give the dropdown a browse class and the data-target or href attribute that correspond with the id attribute of the menu. Note:href value should be prefixed with "#".

The menu can contain items as a normal menu or any desired content. An example markup is below.

Trigger Type

Dropdown can be triggered in two different ways. It could either be by click or hover. The default way a dropdown is triggered is by the click trigger. To trigger the hover trigger, the data-hover attribute should be added to the dropdown. It can be given a value true or no value at all.

Sub Dropdown

A sub dropdown is a dropdown that is meant to exist in a drop menu as an item. It is created with the normal dropdown syntax only that it has the sub class. The default way a sub dropdown works is that it is triggered by the hover trigger (you can easily change that by adding the data-hover=false attribute to the sub dropdown). It display either to the left or right. See example below.

dropdown

two

Dropdown Options

You can customize the behavior of a dropdown using the following options. Options is passed to a dropdown as a value to the data-options attribute of the dropdown in a name:value pair seperated with colon (;). eg data-options ="option1: value1; option2: value2; ..."

Options Default Value Description
delay 0 This specify the delay time in showing or hiding the dropdown (in miliseconds) NB: value should be a number
duration 300 This is the duration of drop menu animation (in miliseconds) NB: value should be a number
view This determine the view position of the dropdown either to be vertical or horizontal. The default value for a normal dropdown is vertical while the default value for a sub dropdown is horizontal
page false It is used to set a dropdown be either a page dropdown or not. If set true, it will make the dropdown menu position to be decided by the mouse position.
contrainWidth false This tell whether to contrain the width of the drop menu to the dropdown width or not.
fluidMinWidth false This tell whether to give the drop menu a minimum with that correspond to the dropdown width or not.
