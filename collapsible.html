Collapsible

A collapsible allows user to toggle the display of sections of content.

A typical collapsible

A typical collapsible consist of a collapsible classed element filled with c-toggler and c-item classed elements.

Places where you can use collapsible

Collapsible can be used with any component. The main thing is understanding that a collapsible contain pairs of toggles (c-toggler) and contents (c-item). You can use a collapsible with panel. See panel.
