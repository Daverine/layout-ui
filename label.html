Label

Easily create nicely looking labels to label and highlight your content.

Definition

A label displays content classification or show short information about a content.

A label is simply created with a text container (eg. span, div, a, etc.) and you then add the label class to the text container previously created. See example below.

a typical label
a link as a label

<div class="label">a typical label</div>
<a class="label">a link as a label</a>
Image Label

A label can be formated to emphasize or wear an image as barge at its left side.

Joe
Veronika

<div class="image label"><img src="<link your image here>"> Joe</div>
<div class="image label"><img src="<link your image here>"> Veronika</div>

Ribbon label

A label can appear as a ribbon attaching itself to an element. This is done by create the label inside the element you want to attach it to and adding the ribbon or alt-ribbon class to the label as shown in the example below.

If the element that a ribbon label is attached to has padding, then a margin-left or margin-right that is equivalent to the element padding-left or padding-right size negatively should be added to the label style depending on either it is a ribbon or an alt-ribbon respectively.

Circular Label

A label can be circular. This is done by adding the circular class to a label.

Note that cicular labels have a square size. and it will not increase even if the content span out. Alternavely, you can use the fully-rounded class to attain a round edge for the label that will increase it size as the content increases.
