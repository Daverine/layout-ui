List

Easily create nice looking lists, which come in different styles.

Introduction

A list is used to group related content. As provided by html, we have the ol, ul and dl. We worked on those provisions to make list more easy and stylable.

ol and ul

OL means ordered list. it is ideally used to group related content giving it a numbering order. We already provided the available ordering type of html list as class for ease access. See example below.

  • The different types that is listed below should be used as class to attain them.
  • To create a list with no indicator, use the no-type class on the list.
decimal-type
  1. Apples
  2. Pears
  3. Oranges
Note that the decimal-type is the default ol type. So you did not need to use any class to use that type. But in a situation whereby the default ordering style has been changed from the decimal-type, the decimal-type class can be added to the list to enable the decimal-type.

UL means un-ordered list. It is ideally used to group related content specifying bullet in front of it. We already provided the available un-ordering type of html list as class for ease access. See example below.

dl (Defination List)

DL means Defination List. Definition lists are made up of a set of terms along with the definitions for each of those terms.

The definition list is created with the <dl> element and usually consists of a series of terms and their definitions. Inside the <dl> element you will usually see pairs of <dt> and <dd> elements. The <dt> is used to contain the term being defined (the definition term). The <dd> is used to contain the definition.

We already syled the dl element to apear more distinct than it was inititially.

Sub-List

We also provide you with a sub-component of list called sub-list. It is used to create a child list. It is just like a typical list just that it has the sub-list class. It is designed to exist in a li element to appear as a child of the li. See example below.

sub-list, when used with ol follows the order of "decimal-type > lower-apha-type > lower-roman-type". And when used with ul follows the order of "disc-type > circle-type > square-type".
sub-list with ol
  1. Animal
  2. Plant
    1. Fruit
      1. Mango
      2. Orange
      3. Pawpaw
    2. Cereal
    3. Tree
  3. Things
sub-list with ul
  • Animal
  • Plant
    • Fruit
      • Mango
      • Orange
      • Pawpaw
    • Cereal
    • Tree
  • Things
<ol>
  <li>your list content here...</li>
  <li>your list content here...
    <ol class="sub-list">
      <li>your sub-list content here...</li>
      <li>your sub-list content here...</li>
      <li>your sub-list content here...</li>
    </ol>
  </li>
  <li>your list content here...</li>
</ol>
You can also create a ul sub-list as in the same way in the example above. A sub-list can be as many as you want. Just follow the format above.

Icon bullet List

We also provide a utility that can enable you to use a specified icon as the numbering indicator of a list. This is done by giving a list (ol, ul or a container with .item classed children) the icon-bullet class and place your icon as the first child of each item (a li element or an item classed element) in the list.
