Vertical Divider

A vertical divider can segment content vertically.

Vertical dividers requires that its parent container has a style of position: relative
Vertical dividers have an absolute position. You will need to specify the left value for it in css to position the divider. Eg. left: 40px;
We provide you with some predefined classes to position a vertical divider. Those classes can only be applied on the dividers' container in form of dividing it to a number(a number from 2 - 12) of equal part as desired. The classes are in the following format: divided-to-x where x in the format is a number form 2 to 12.
Note that to use the predefined classes provided to posiiton your divider, the vertical divider(s)' element name must be unique from it siblings element. Ie. all dividers should have same element name and all other siblings element should have another element name. For instance if the dividers' siblings element is div the divider(s) can be span.

Stackable Divider

A stackable divider appears as a vertical divider from any screen size larger that 730px (medium screen) and as horizontal divider from a screen size below 730px. It can also be positioned by the predefined classes provided for vertical dividers only that when it get to screen width of 730px and below, it changes to horizontal divider.

Compact and Loose Divider

A divider can apear compact (ie. fitted without any space above or below it) or loose (ie. with large margins to divide sections of content). Add the compact or loose class to a divider as either to produce a compact or a loose divider.

Transparent Divider

A transparent divider divide contents without creating a dividing line. Just add the transparent class to a divider to produce one.

Clearing Divider

A divider clear or balance a floated content above it. Just add the clearing class to a divider to produce one.
