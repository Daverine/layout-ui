Image

Easily Style, size and position an image with our provided predefined classes.

Introduction

An image is a graphic representation of something. We create the image class to enable the use of our predefined classes to style an image. You can specify an img or svg as an image or use them as a child element of an image classed element.

Variations

A image can be formated to appear in various form.

Avatar

An image may be formatted to appear inline with text as an avatar. To create an avater, add the avatar class to an image.

Username
Bordered

An image may include a border to emphasize the edges of white or transparent content. Add the bordered class to an image to give it a border.

Rounded

An image can have round edge. Add the rounded class to an image to make it edge round.

Circular

An image may appear circular. Add the cicular class to an image to make it circular.

Image positioning

Image can be formated to appear in different positions.

Vertical Aligning

Centered

An image can appear centered in a content block. Just add the centered class to the image.

Floated

An image can sit to the left or right of other content. Add the l-floated class to an image to float it to the left and the r-floated class to float it to the right.

Image size

Unless a size if specified, images classed with the image class will use the original dimensions of the image, with a maximum width of its container size max-width: 100%.

We provide some predefined size to size an image. See it in the below table and example.

Note that the height of an image is set to auto making the height change in relation to the image width. So the image width is the size specified in the table.
Class Name Size
Mini 100px
Small 200px
Medium 400px
Big 600px
Huge 800px
