Create a fully responsive grids and containers.

Container

A container is an element designed to contain page elements to a reasonable maximum width based on the size of a user's screen. This is useful to couple with other UI elements like grid or menu to restrict their width to a reasonable size for display.

Container Sizes

Containers are designed to responsively adjust their maximum width based on the size of the screen they are appearing. It is created by adding the container class to a block element.

Grid

A grid is used to harmonize negative space in a layout.

Using a grid makes content appear to flow more naturally on your page. It is created by adding the grid class to a block element. Grids divide horizontal space into unit(s) called "columns". We choose as default a maximum of 12 columns per row. All columns in a grid are provided with three (3) methods to size them. 1. To specify the size equally, 2. specify the size individually, and 3. specify the size manually.

Note that if columns' size is not specified in a grid, the grid system will chooses an arbitrary column count to allow per row as the viewport and columns' content allows.

Individually size columns in a grid

To individually size columns in a grid, all columns in that grid must specify their width as proportion of the total available row width. These is done by giving each columns in the grid a class in x-width format where x is a number from 1-12 " which denote the width as x of 12 of the row width. Dont forget that x is a number from 1-12.

Gutter

Grids automatically create horizontal gutter between columns. To avoid the gutter you will need to add the no-gutter class to the grid.

Source Ordering

You can change the display order of the columns to keep a specific column order in the source code. Add one of the *-push classes to move the column to the right and add one of the *-pull classes to move a column to the left. This allows you for example to flip the columns' display order for wider viewports. We also provide the *-offset class to offset columns which enables creating additional space between them.

Note: * in the classes metioned above is a number from 1 to 12.

Note that the number in front of -pull, -push and -offset is any number between 1 and 12. The size of the pull, push and offset is determined by the size of the parent width divided by 12 and multiplied by the number specified ie. if 2-pull is used on an element then (parentWidth/12) * 2 is the size used for the pull.
