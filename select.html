Select Dropdown

Define a select-box-like component. Styleable, customizable with added features.

Introduction

A select dropdown is just like a regular dropdown only that it can be used as a form component just like a select box. Another thing is that select dropdown also support full keyboard control eg navigation, closing with escape, etc. View example below.

Usage

Building a select dropdown is as much as building a regular dropdown. You will create a div element that has the select dropdown class and a <div> menu created inside the dropdown div given the drop menu class. Your drop menu can now be filled with your desired items in these format: <div class="item" data-value="your value">item name</div>. Another thing is that you must create an hidden input inside the dropdown. This is used to store the value for the item selected by end-user, and this input will be responsible for the value that will be submitted when you use the select dropdown in a form. View the code for a typical select dropdown below.

<div class="select dropdown">
   <div class="placeholder">Select an Item</div>
   <i class="icon i-menu-down"></i>
   <input type="hidden" name="your desired name">
   <div class="drop menu">
     <div class="item" data-value="1">item 1</div>
     <div class="item" data-value="2">item 2</div>
     <div class="item" data-value="3">item 3</div>
     <div class="item" data-value="4">item 4</div>
     ...
   </div>
</div>
  • You can optionally add a placeholder to your select dropdown as shown in the code above.
  • If the data-value attribute is ommited in the drop menu items, the content text of the item will be used as value.
  • You can pre-select an item by adding the active class to the item you want to pre-select.

Tweaks and Utilites

A select dropdown can be tweaked to fit appear in various enviroment and make it nice looking.

Fix Indicator padding in select dropdown

You might want to remove right padding of the select dropdown reserved for the menu-down indicator if you are not using the indicator. Add the no-indicator class to your select dropdown to remove the padding.
