Modal

Create modal dialog with diffrent styles and transistions.

Definition

A modal displays content that temporarily blocks interactions with the main view of a site.

Introduction

A modal is created with a modal classed div. It can contain any content. For a modal to work, it requires something to trigger it. The modal-trigger class is intialized on a click trigger. So any element that has the modal-trigger class and an href or data-target attribute linking to the id of a modal will trigger the modal on click.

Click on the button below to see how a modal works.

It is advicible to make your modal element a direct child of the <body> tag in other to avoid any unexpected display of the modal.
MODAL
<div class="modal" id="modal1">
  ...
</div>

<a class="modal-trigger" href="#modal1">modal trigger</a>
or
<div class="modal-trigger" data-target="modal1">modal trigger</div>
Self Scroll Modal

If the content of a modal requires scrolling, the modal overlay will be the scroller. ie. The overflowed modal is wraped in an overlay that can scroll it. But if you want the modal itself to be scrollable, you add the self-scroll to the modal. In addition, the self-scroll class will centralilze the modal in the overlay.

Click the buttons below to compare.

NORMAL MODAL MODAL SELF SCROLL STYLE
<div class="self-scroll modal" id="modal1">
  ...
</div>
Bottom Sheet Modal

A bottom sheet modal pops-up from the bottom of the screen and display it content at the bottom of the screen. To create a bottom sheet modal, add the bottom-sheet class to a modal.

MODAL BOTTOM SHEET STYLE
<div class="bottom-sheet modal" id="modal1">
  ...
</div>
Full Screen Modal

A Full screen modal makes the modal fill the entire screen. It is compostory to have a close button to close the modal. It is created by adding the full-screen class to a modal.

FULL SCREEN MODAL
<div class="full-screen modal" id="modal1">
  ...
</div>
Animation

We provide six different animation classes to open and close a modal. They are fade, all-scale, horizontal-scale, vertical-scale, horizontal-flip and vertical-flip. A disired animation class should be added to a modal.

Click on the buttons aside the animation name to preview.

Class name Preview Button
fade Fade
all-scale All Scale
horizontal-scale Horizontal Scale
vertical-scale Vertical Scale
horizontal-flip Horizontal Flip
vertical-flip Vertical Flip
Modal Initialization

You can open a modal using a trigger:

The trigger must have either a href or data-target attribute linking to the id of the modal to open. If you are using the href attribute, the value must be prefixed with #.
$(document).ready(function(){
// the "href" or "data-target" attribute of .modal-trigger must specify the modal ID that wants to be triggered
$('.modal-trigger').modal(options);
});

You can also open modals programatically, the below code will make your modal open.

$('#modal1').openModal(options);

You can also close them programatically:

$('#modal1').closeModal();
NB: the options parameter is optional
Modal Options

You can customize the behavior of each modal using the following options. Options can be passed to a modal as a parameter when it is initialized or as a value to the data-options attribute of the modal in an object format. ie. name:value pair seperated with (,) eg{ option: value, ... }

Options Default Value Description
closeOnEsc true This option controls the ability to close a modal on Escape Key press.
closeOnLayerClick true This option controls the ability to close a modal on the click of the faint-transparent layer that covers the page.
blurring false This tell wether to blur the page on when modal is opened.
duration 400 This is the duration of modal animation (in miliseconds) NB: value should be a number
ready undefined This option can either be undefined or a function called to called when the modal finished opening.
complete undefined This option can either be undefined or a function called to called when the modal finished closing.
