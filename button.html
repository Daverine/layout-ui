Button

Easily create nicely looking buttons, which come in different styles.

Definition

A button indicates a possible user action.

Button Usage

To apply this component, add the button class to any element. Now you have a button. Add the disabled class or attribute to the created button to disable it.

Although any tag can be used for a button, it will only be keyboard focusable if you use a <button> or <a> tag or you add the property tabindex="0". Keyboard accessible buttons will preserve focus styles after click, which may be visually jarring.

Animated Button

A button can animate to show hidden content. .It is a button that has a class of one of the below button namespace. It contains a <div> with class visible-content and another <div> with class hidden-content.

The button will automatically sized according to the widest content of the button.
<button class="namespace button">
   <div class="visible-content">...</div>
   <div class="hidden-content">...</div>
</button>
